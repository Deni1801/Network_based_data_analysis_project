This project analyzes gene expression data to distinguish carcinoma-associated fibroblasts (CAFs) from normal fibroblasts (NFs) in non-small cell lung cancer.

It uses exploratory analysis, PCA, K-means clustering, and supervised machine learning models including Random Forest, LDA, and LASSO logistic regression. Functional enrichment analysis is performed with g:Profiler and pathfindR to identify biological pathways associated with CAFs.

The results show that supervised models, especially Random Forest and LDA, classify CAFs and NFs effectively, while enrichment analysis highlights extracellular matrix remodeling, collagen biosynthesis, and developmental signaling as key CAF-related processes.
